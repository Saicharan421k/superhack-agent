# SuperHack 2025: AI Agent for Intelligent Ticket Triage

This repository contains the prototype for our SuperHack 2025 submission.

## Core Features
- AI-Powered Ticket Analysis using Amazon Bedrock
- Automated Root Cause Hinting & Action Plan Generation
- User Self-Service Portal for Ticket Submission & History
- MSP Agent View with Enriched Asset & Alert Data

## Tech Stack
- Backend: Python / Flask on Amazon EC2
- Frontend: HTML, CSS, JavaScript
- AI: Amazon Bedrock (Claude 3 Sonnet)
- Database: Amazon S3 (Simulated)
